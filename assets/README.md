# Mapa de archivos del sitio

Estructura canónica para imágenes y documentos del sitio web de Fundación N.A.H.U.E.L.

## Marca institucional

Usar esta carpeta para logos, isotipos y recursos gráficos de identidad:

```text
assets/img/brand/
```

Archivo recomendado:

```text
assets/img/brand/logo-nahuel.jpg
```

## Registros de actividades

Usar esta carpeta para fotografías de actividades y bitácoras visuales:

```text
assets/img/registros/
```

Para la actividad REG-2026-001:

```text
assets/img/registros/reg-2026-001-restauracion-mural-val/
```

Nombres recomendados:

```text
2026-02-17_reg-2026-001_val_01_antes-detalle.jpg
2026-02-17_reg-2026-001_val_02_antes-contexto.jpg
2026-02-17_reg-2026-001_val_03_despues-general.jpg
2026-02-17_reg-2026-001_val_04_despues-muro-completo.jpg
2026-02-17_reg-2026-001_val_05_cierre-territorial.jpg
```

## Evitar

No usar carpetas con espacios finales, tildes, mayúsculas innecesarias o nombres genéricos como:

```text
assets/img /Actividades/
assets/img /Actividades /
```

No usar nombres informales en archivos públicos. Mantener nombres descriptivos, institucionales y estables.
